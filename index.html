Responsive Facebook Comments - jQuery plugin

Responsive comments for facebook that spans mobile, tablet and desktop
download .ZIPdownload .TGZ

This plugin was made to take the guesswork out of making the facebook comments responsive and able to work on mobile phones without loading issues.

What ended up working was to turn mobile off and make the container div responsive (based off of percent) then refresh the content on page load. Works great! Give it a try.

Initialization

To initialize the plugin simply call: 



// Put this inside the jQuery document ready statement
    $('#container').fbresponsivecomments({
        appId:'your_appId_here',
        url: 'your_domain_here' 
    });



// Put this in the body where you want the comments to show up
<div id="container">
     <div class="fb-comments" data-href="YOUR PAGE URL HERE" data-num-posts="2" mobile="false"></div>
</div>

Options

  • 'appId': 'your_appId_here',
  • 'url': 'your_url_here',
  • 'posts' : '5'
  1. appId required : the appId Facebook gives you when you create an app found at Facebook Apps
  2. url required : the domain it will reside on. Ex. http://www.example.com
  3. posts optional : default: 5 - the amount of posts to show initially. If there are more comments than the default it will produce a button that reads "load more posts"
